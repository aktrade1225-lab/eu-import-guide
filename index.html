<!-- Application Structure Plan:

This SPA uses a "Dashboard/Exploratory" layout. Instead of a long, scrolling document, the user is presented with a central interactive hub.

1. Hero & Intro: Sets the context and explains the guide's purpose.

2. Interactive Category Explorer: The core interaction. Users click on a product category (Food, Cosmetics, Electronics) to dynamically load its specific compliance processes, key difficulties, and a visual difficulty gauge. This reduces cognitive load by only showing relevant information.

3. Comparative Matrix: A side-by-side visual table allowing users to quickly contrast the requirements of all three categories.

4. Strategic Support: Concluding section detailing Tammi's solutions to the highlighted problems.

This structure was chosen to make complex regulatory information digestible, task-oriented, and easy to navigate for business partners. -->

<!-- Visualization & Content Choices:

1. Difficulty Gauge (Chart.js Doughnut): Report Info -> Difficulty Rating (1-5) -> Goal: Visually communicate risk/barrier to entry -> Interaction: Updates dynamically when a category is clicked -> Justification: A half-circle gauge instantly conveys "how hard is this" better than text stars.

2. Category Cards (HTML/CSS/JS): Report Info -> 3 Main Categories -> Goal: Organize content -> Interaction: Clickable selectors -> Justification: Keeps UI clean; users focus on one product type at a time.

3. Comparison Matrix (Tailwind Grid): Report Info -> Summary Table -> Goal: Compare cross-category requirements -> Interaction: Hover effects for readability -> Justification: Best way to show qualitative comparisons side-by-side.

Confirming NO SVG/Mermaid used. HTML/CSS and Unicode emojis are used for all iconography. -->

function selectCategory(categoryId) {

// Update buttons state

document.querySelectorAll('.category-btn').forEach(btn => {

btn.classList.remove('active');

});

document.querySelector(`[data-category="${categoryId}"]`).classList.add('active');



currentCategory = categoryId;

updateUI();

}



function updateUI() {

const data = reportData[currentCategory];



// Update Text Elements

document.getElementById('difficulty-desc').innerText = data.difficultyDesc;

document.getElementById('difficulty-score').innerText = data.difficulty;

const labelEl = document.getElementById('difficulty-label');

labelEl.innerText = data.difficultyLabel;

labelEl.style.color = data.chartColor;

labelEl.style.borderColor = data.chartColor;



// Update Process List

const processList = document.getElementById('process-list');

function updateChart(difficultyScore, color) {

const ctx = document.getElementById('difficultyChart').getContext('2d');

// For a gauge from 1 to 5

const maxScore = 5;

const remainder = maxScore - difficultyScore;



const chartData = {

labels: ['難度', ''],

datasets: [{

data: [difficultyScore, remainder],

backgroundColor: [color, '#e2e8f0'], // Value color, empty color

borderWidth: 0,

circumference: 180, // Half circle

rotation: -90 // Start from left

}]

};



const config = {

type: 'doughnut',

data: chartData,

options: {

responsive: true,

maintainAspectRatio: false,

cutout: '75%', // Make it thin

plugins: {

legend: { display: false },

tooltip: { enabled: false } // Disable tooltip for gauge

},

animation: {

animateScale: false,

animateRotate: true,

duration: 800,

easing: 'easeOutQuart'

}

}

};



if (difficultyChartInstance) {

difficultyChartInstance.data = chartData;

difficultyChartInstance.update();

} else {

difficultyChartInstance = new Chart(ctx, config);

}

}



// Initialize on load

window.addEventListener('DOMContentLoaded', () => {

updateUI(); // Load default (food)
